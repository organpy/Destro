# Destro

A cross-platform desktop application for synchronized audio playback. Destro allows multiple users to listen to YouTube, YouTube Music, and SoundCloud streams simultaneously in real-time.

## Downloads

Pre-compiled binaries for Windows and Linux are available on the [Releases](https://github.com/Aymenzito/destro/releases) page. 

* **Windows:** `.exe` installer
* **Linux:** `.AppImage` / `.deb`

## Features

* **Real-Time Synchronization:** Play, pause, and seek events are synchronized across all connected clients with minimal latency.
* **Multi-Service Support:** Streams audio directly from YouTube, YouTube Music, and SoundCloud.
* **Native Desktop Integration:** Built specifically for Windows and Linux environments.
* **Open Source:** The codebase is fully open for review, contribution, and auditing.

## Local Development

To build and run Destro from source:

1. Clone the repository:
   ```bash
   git clone [https://github.com/Aymenzito/destro.git](https://github.com/AymennX/Destro.git)
   cd Destro
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm run start
   ```

## Tech Stack

* **Desktop Framework:** Electron JS
* **Core:** JavaScript (JS) & React
* **Styling:** CSS
* **Build Tool:** Vite
